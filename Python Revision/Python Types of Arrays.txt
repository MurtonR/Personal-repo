List is a collection which is ordered and changeable. Allows duplicate members.  ["Joe", "Biden"]

Tuple is a collection which is ordered and unchangeable. Allows duplicate members. ("Joe, Biden")

Set is a collection which is unordered, unchangeable*, and unindexed. No duplicate members. {"Joe", "Biden"}

Dictionary is a collection which is ordered** and changeable. No duplicate members. 
        
{
"Brand" : "Renault",
"Model" : "Reliable",
"Colour" : "Red",
}
